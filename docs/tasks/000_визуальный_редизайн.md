## 1. Общий стиль — типографика и отступы

**Проблема:** Шрифт `Geologica` хороший, но некоторые размеры слишком мелкие (11–11.5px) — плохо читается.

**Что изменить в `:root`:**

```css
/* Текущие токены оставить, добавить/уточнить: */
--radius-sm: 6px;
--radius-md: 10px;
--radius-lg: 14px;
--radius-xl: 18px;

/* Тени сделать мягче и современнее */
--shadow: 0 1px 4px rgba(0,0,0,0.06), 0 4px 16px rgba(107,63,160,0.06);
--shadow-lg: 0 4px 24px rgba(107,63,160,0.12), 0 1px 4px rgba(0,0,0,0.08);
```

**Типографика — минимальные размеры:**
- Нигде ниже `12px` (сейчас есть `10px`, `10.5px`, `11px` — это слишком мелко)
- Основной текст `td` поднять с `13.5px` → `14px`
- Лейблы форм с `12px` → `12.5px`
- Хинты и мета-текст: минимум `12px`
